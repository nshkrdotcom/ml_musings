**Breakdown of all symbols** in the backpropagation equations shown in the image.

These equations come from the standard derivation of **backpropagation** (the algorithm used to compute gradients for training neural networks). They show how to efficiently calculate how the cost changes with respect to the weights and biases by propagating errors backward from the output layer.

### The Four Equations (with context)

1. **Output layer error**  
   \[
   \delta^L = \nabla_a C \odot \sigma'(z^L)
   \]

2. **Hidden layer error (backpropagation step)**  
   \[
   \delta^l = \bigl( (w^{l+1})^T \delta^{l+1} \bigr) \odot \sigma'(z^l)
   \]

3. **Gradient w.r.t. biases**  
   \[
   \frac{\partial C}{\partial b_j^l} = \delta_j^l
   \]

4. **Gradient w.r.t. weights**  
   \[
   \frac{\partial C}{\partial w_{jk}^l} = a_k^{l-1} \delta_j^l
   \]

### Symbol Glossary (complete breakdown)

| Symbol                  | Meaning                                                                 | Type          | Notes / Layer |
|-------------------------|-------------------------------------------------------------------------|---------------|---------------|
| \(\delta^L\)            | Error / sensitivity of the **output layer**                             | Vector        | Output layer \(L\) |
| \(\delta^l\)            | Error / sensitivity of layer \(l\)                                      | Vector        | Any layer \(l\) |
| \(\delta_j^l\)          | The \(j\)-th component of \(\delta^l\) (for neuron \(j\) in layer \(l\)) | Scalar     | Neuron-specific |
| \(\delta^{l+1}\)        | Error from the **next** layer (used to compute current layer's error)   | Vector        | Layer \(l+1\) |
| \(\nabla_a C\)          | Gradient of the cost \(C\) w.r.t. the **output activations** \(a^L\)    | Vector        | Output layer only |
| \(C\)                   | **Cost / loss function** (e.g. mean squared error, cross-entropy)       | Scalar        | Overall network |
| \(\odot\)               | **Hadamard product** (element-wise multiplication)                      | Operation     | Vector × vector |
| \(\sigma'\)             | Derivative of the **activation function**                               | Function      | Usually sigmoid, ReLU, etc. |
| \(\sigma'(z^L)\)        | Derivative of activation evaluated at output layer's pre-activations    | Vector        | Output layer |
| \(\sigma'(z^l)\)        | Derivative of activation evaluated at layer \(l\)'s pre-activations     | Vector        | Hidden layer |
| \(z^L\)                 | **Weighted input** (pre-activation) of the output layer                 | Vector        | \(z^L = w^L a^{L-1} + b^L\) |
| \(z^l\)                 | **Weighted input** (pre-activation) of layer \(l\)                      | Vector        | \(z^l = w^l a^{l-1} + b^l\) |
| \(w^{l+1}\)             | Weight matrix connecting layer \(l\) to layer \(l+1\)                   | Matrix        | Transposed when backpropagating |
| \((w^{l+1})^T\)         | Transpose of the weight matrix from the next layer                      | Matrix        | Used to propagate error backward |
| \(w_{jk}^l\)            | Weight from neuron \(k\) in layer \(l-1\) to neuron \(j\) in layer \(l\) | Scalar     | Individual weight |
| \(b_j^l\)               | Bias of neuron \(j\) in layer \(l\)                                     | Scalar        | Individual bias |
| \(a_k^{l-1}\)           | Activation (output) of neuron \(k\) in the **previous** layer           | Scalar        | From layer \(l-1\) |
| \(a^l\)                 | Activation vector of layer \(l\) (after applying \(\sigma\))            | Vector        | \(a^l = \sigma(z^l)\) |
| \(\frac{\partial C}{\partial b_j^l}\) | How much the cost changes when we change bias \(b_j^l\)          | Scalar        | What we want for gradient descent |
| \(\frac{\partial C}{\partial w_{jk}^l}\) | How much the cost changes when we change weight \(w_{jk}^l\)     | Scalar        | What we want for gradient descent |
| \(L\)                   | Index of the **final / output layer**                                   | Integer       | Last layer |
| \(l\)                   | Index of a **general layer** (usually a hidden layer)                   | Integer       | \(l = 1, 2, \dots, L-1\) |
| \(j\)                   | Neuron index in the **current** layer                                   | Integer       | Row index in weight matrix |
| \(k\)                   | Neuron index in the **previous** layer                                  | Integer       | Column index in weight matrix |

### Quick Intuition

- \(\delta^l\) tells you **"how much this neuron contributed to the error"**.
- The first equation computes it directly at the output using the cost gradient.
- The second equation **propagates** that error backward through the network.
- The last two equations show that once you have \(\delta^l\), the gradients for biases and weights are almost trivial to compute.

This is the mathematical heart of why backpropagation is efficient: you only need one forward pass + one backward pass to get **all** the gradients. 

The little cartoon character at the bottom is just flavor (common in explanatory videos about backprop). The colored highlighting in the original image (yellow for deltas, red for \(C\), green for \(\sigma\), blue for weights, etc.) is just to make the variables stand out visually.
